# Telegram Bot

## Run the bot

**macOS / Linux:** use the helper script (creates venv, installs requirements, runs the bot):

```bash
./run.sh
```

**Windows:** run `run_bot.bat`.

---

**Manual run** (if you prefer to set up yourself):

1. Install requirements: `pip install -r requirements.txt`
2. Run: `python bot.py`
